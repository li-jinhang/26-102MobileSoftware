# 模拟器卡片边缘流光设计

## 目标

将当前独立的细条光效替换为与工作台统计卡片、知识详情标题卡片融为一体的边缘流光，使 HarmonyOS 7 模拟器能够稳定演示明显且不占额外纵向空间的沉浸光感。

## 视觉设计

- 删除工作台顶部独立 `HdsAccent` 占位区域。
- “你好，张晓宁”统计卡片保留原有内容和尺寸，增加浅蓝边框、轻微阴影和顶部双向移动光带。
- 蓝色光带从左向右移动，绿色光带从右向左移动，光带高度保持在 3-5vp，不遮挡文字。
- 知识详情标题卡片使用相同边缘流光，形成一致的 HarmonyOS 展示语言。
- 不增加功能说明文字，不改变工作台、详情页和底部导航的交互。

## 技术方案

- 新增可复用 `EdgeLightPanel` 容器组件，使用 ArkUI `Stack`、绝对定位、平移动画和透明度实现模拟器流光。
- `EdgeLightPanel` 接收内容 Builder、高度和强调颜色，不把业务数据放入光效组件。
- 继续保留 `HdsVisualComponent`，但 HDS 不作为可见效果的唯一来源；模拟器始终渲染 ArkUI 边缘光带。
- 动画生命周期随组件出现启动、消失停止，避免定时器或动画在页面离开后继续运行。

## 验收标准

1. 工作台不再显示独立空白光感矩形或细进度条。
2. 统计卡片边缘可以看到蓝绿光带沿相反方向移动。
3. 光带不遮挡问候语、部门信息和三个统计数字。
4. 知识详情标题卡片显示相同风格光效。
5. 模拟器安装后动画持续运行，页面切换和返回不崩溃。
6. 主 HAP 和现有 Hypium 测试继续通过。
