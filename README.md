# 智协库

面向公司内部的知识库与流程协作鸿蒙移动应用课程设计项目。

本项目基于 HarmonyOS + ArkTS + ArkUI 实现，围绕“查知识”和“办流程”两个核心场景，构建了一个适合课程展示的移动端协作应用原型。当前版本以本地模拟数据驱动，重点展示知识库、流程审批、消息提醒、角色权限和安全确认等完整业务闭环。

## 项目简介

在企业内部办公场景中，员工经常需要先查制度、再提流程、再等审批。传统系统往往将知识库、待办消息和流程入口分散在不同位置，使用成本较高。

“智协库”将以下能力整合到同一个鸿蒙应用中：

- 工作台：集中展示待办、快捷流程、最近知识和消息提醒
- 知识库：支持分类浏览、详情查看、收藏、有用反馈和纠错建议
- 流程中心：支持请假、报销、采购、权限申请等典型流程
- 智能助手：基于关键词匹配返回知识摘要并推荐可发起流程
- 安全治理：高风险流程触发二次确认，并写入操作日志

## 技术栈

- HarmonyOS
- ArkTS
- ArkUI
- Hvigor 构建体系
- 本地 Mock 数据驱动页面与状态流转

## 当前已实现功能

### 1. 登录与角色体验

- 支持模拟账号登录
- 根据角色展示不同的操作视角
- 支持退出登录

当前内置角色：

- 普通员工
- 审批人
- 系统管理员
- 知识管理员

### 2. 工作台

- 展示当前用户身份与部门信息
- 展示待办数量、消息入口、知识入口
- 支持从首页快速进入智能助手、消息页和流程页

### 3. 知识库

- 按分类浏览知识条目
- 查看知识摘要、标签、版本和更新时间
- 打开知识详情
- 收藏知识
- 标记“有用”
- 提交纠错建议
- 展示知识与流程的关联关系

### 4. 智能助手

- 支持自然语言输入问题
- 基于关键词匹配返回知识摘要
- 推荐相关流程
- 支持从问答结果跳转到知识详情或流程发起
- 保留最近问答记录

### 5. 流程协作

- 内置请假申请、报销申请、采购申请、权限申请四类模板
- 支持流程表单填写与基础校验
- 支持流程提交、审批、驳回、状态流转
- 展示我的流程与审批待办
- 展示审批意见与审批记录
- 将制度知识与流程模板联动展示

### 6. 风险控制与日志

- 报销金额超过 5000 元触发二次安全确认
- 采购金额超过 10000 元触发二次安全确认
- 系统管理员可处理高风险确认
- 关键操作生成通知与审计日志

## 核心文件说明

- `entry/src/main/ets/pages/Index.ets`：应用主页面与核心状态管理
- `entry/src/main/ets/mock/MockData.ets`：模拟用户、知识、流程、消息、审计日志数据

- `entry/src/main/ets/common/services/`：知识、消息、助手相关业务处理逻辑
- `entry/src/main/ets/features/`：各业务页面模块
